# Boris Hanicotte Portfolio

Static personal portfolio website for GitHub Pages.

Live site: https://borishnt.github.io/me/index.html

## Description

A small personal portfolio presented as a modern flat desktop interface inspired by early graphical operating systems. It uses window panels, title bars, file/archive metaphors and static project data rendered with vanilla JavaScript. The About page is a separate text-document experience with controlled scroll-based corruption.

## Run Locally

Open `index.html` directly in a browser.

No install step is required.

## GitHub Pages

This repository is ready for GitHub Pages. Push the files, enable Pages from the repository settings, and serve the site from the repository root.

## Technologies

- HTML
- CSS
- Vanilla JavaScript
- Static files only
